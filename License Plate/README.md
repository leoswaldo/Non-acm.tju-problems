# License Plate


### Problem Description
Given a License Plate, find all coincidences that matches with all characters of
it from a list of words (not case sensitive), except dashes and print the number
of coincidences

### Example

### Sample input
license_plate = "PY-34-THON"

words_list = ["ASDF3490PASaTh", "TH3214Py0on", "TH3214Py0N", "TH3214Py0o"]

### Sample output
output = 1
