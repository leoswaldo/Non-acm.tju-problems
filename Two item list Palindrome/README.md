#Two Items List Palindrome


###Problem Description:

Code an script which will find if joining two elements of the input form a
palindrome for example: given a list ['food', 'deal', 'doof'], your script
should find that food + doof form a palindrome


###Input

A list of words separeted by a space


###Output

A message showing if joining words generates a palindrome


###Input example

    ./palindrome.py  food state deal doof

###Output example

    food + doof is palindrome
    doof + food is palindrome
