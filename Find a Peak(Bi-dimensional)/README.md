Find a peak element
=========


Description
----
Given an matrix of integers. Find a peak element in it. An array element is peak
if it is NOT smaller than its neighbors (up, left, right, down). For corner
elements, we need to consider only two neighbors.

For example
----

Matrix

    3, 6, 1, 5, 7
    9, 3, 1, 8, 3
    2, 4, 5, 7, 0
    4, 8, 2, 6, 3

There are five peak elements in positions:  [0, 1], [0, 4], [1, 0], [1, 3], [3, 1]


Note / Answer
----
> You need to print any peak element.
